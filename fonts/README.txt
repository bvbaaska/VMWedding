Сюда нужно положить файлы шрифта Marianna.

Шрифт используется для всех заголовков и названий разделов.
Marianna — коммерческий шрифт (Contrast Foundry), его нет в
бесплатных Google Fonts, поэтому файлы нужно добавить вручную.

Нужные файлы (имена должны совпадать точь-в-точь):

    Marianna-Regular.woff2
    Marianna-Regular.woff
    Marianna-Bold.woff2      (необязательно, для жирного начертания)
    Marianna-Bold.woff       (необязательно)

Как добавить:
1. Получите файлы шрифта Marianna (купите/скачайте лицензию).
2. Если есть только .ttf или .otf — сконвертируйте в .woff2/.woff
   на любом онлайн-конвертере (например transfonter.org).
3. Переименуйте по списку выше и положите в эту папку (wedding/fonts/).
4. Скажите мне — я закоммичу и запушу.

Пока файлов нет, сайт автоматически использует запасной шрифт
(Cormorant Garamond / Georgia), так что вёрстка не ломается.

Основной текст использует шрифт Vollkorn — он уже подключён
из Google Fonts, ничего добавлять не нужно.
