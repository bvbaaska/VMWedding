Сюда нужно положить файл с песней:

    marry-you.mp3

Это Bruno Mars - Marry You.

Где взять mp3:
1. Скачайте песню из любого источника (YouTube → mp3 конвертер,
   или из вашей музыкальной библиотеки).
2. Переименуйте файл ровно в:  marry-you.mp3
3. Положите его в эту папку (wedding/music/).
4. Загрузите в GitHub (скажите мне — я закоммичу и запушу).

После этого музыка заработает на сайте автоматически.
